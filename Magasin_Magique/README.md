# Module Tests
Réalisation d'exercices avec le projet MagasinMagique :  


## Objectif : 
implanter une nouvelle catégorie de produits à vendre, et réorganiser le code afin de le rendre plus maintenable.  

## Problème : 
le projet ne comportant pas de tests pré-établis, il faut s'assurer de ne pas casser le code déjà en place.  

## méthodologie :
Réalisation d'un Golden Master : un fichier créé avant la modification du code, comprenant les résultats de différents cas spécifiques. Après modification du code, un nouveau fichier est généré, puis comparé au premier.  
Les résultats des deux fichiers doivent être identiques.
